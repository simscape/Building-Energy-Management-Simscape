Setup Building Model for BEMS Controller Design

Copyright 2024 - 2025 The MathWorks, Inc.
In the workflow Simulate Building Energy Management System (Workflow/SimulateBuildingEnergyManagement.mlx), you will evaluate heat pump performance under different building loading conditions. The workflow will setup the model to enable controller design for realistic scenarios. You can evaluate the HVAC concepts such as a room radiator and/or the underfloor piping. To open and run the workflow, enter at the MATLAB command prompt.
edit SimulateBuildingEnergyManagement.mlx
This analysis uses BuildingModelWithSolarLoadAndHVAC SLX model for analysis (Models/BuildingModelWithSolarLoadAndHVAC.slx).
