Rooms

Rooms are the basic blocks in creating a building definition using MATLAB utilities in the workflow Create Building Model Part Using Utilities. Rooms are defined as rectangluar, or a union of rectangular shapes. The rooms are created as a part of the floor-plan of an elevation.
The picture above shows a few examples of rooms that are permitted and those that are not permitted. As can be seen in figure, curved walls is not supported. Rooms must be rectangles, squares, or a union of such basic shapes. For a non-rectangular or square room, you must specify multiple rooms that are rectangular and eliminate the common walls betweem them. The room connectivity is calculated automatically by the MATLAB utilities and wall lengths estimated.
The picture above shows the room connectivity data that is automatically calculated in the code. There are 5 rooms in the above example and the walls are automatically identified as being external or internal.
By default, each room is modelled as a thermal mass connected to 4 walls, a roof, and a floor element. There are multiple custom library blocks provided in this project which can help you create rooms with surrounding walls, floors, and roofs. You may also add HVAC components like radiator and under floor piping to the rooms. Four composite components provide rooms models with these HVAC elements.

Apartments

Multiple rooms make an apartment. These rooms must be aligned along or have common walls with one or more rooms. Each room has a unique identification provided by it's apartment and room number. All MATLAB utilities to create and parameterize buildings are based on the apartment and room numbers. The building part stores data as a struct which is the apartment and room defintion for the building model.

FloorPlans

Floorplan is a collection of one or more apartments. The floorplan is the final definition of the building layout based on which the elevation is created.

Building

A building is the floorplan data extended in vertical direction with a specified number of floors. Each floor os of the same height. The building can be oriented in any direction. The geographical location and building orientation data are used to evaluate the solar radiation falling on the external walls and roof of the building.
East,west, north, and south are represented on the x-y plane. All walls (Solar Radiation on Surface in combination with the Wall Selector block, or the External Wall block) require this orientation information to estimate the solar load on the building.
